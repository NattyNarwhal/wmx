NAME

wmxc — control a wmx instance

SYNOPSYS

wmxc [-p port] [-q] command [arguments]

DESCRIPTION

wmxc is a small program that interacts with a wmx(1) instance with the remoting feature enabled. This allows you to probe and control wmx from the shell, allowing you to control it from SSH sessions, scripts, and so on.

wmx listens on port 6999 (can be changed) and can be controlled via this port, but it can be harder to use interactively - there is no shell parsing or readline to make it nice, and it harder to automate. wmxc should solve these problems.

The options are as follows:

-p port
A custom port to use. By default, wmx listens on 6999.
-q
Don't output anything.

USING WMXC

wmxc expects a command and optionallly, arguments. The commands are as follows:
channel [number]
Shows the current channel number. If a channel is given, switches to that channel.
clients
Show the current windows and their IDs.
exit
Exits wmx.
hide window
Hides the window.
label window string
Changes the window title/label.
lower window
Lowers the window.
maximize window number
Maximizes the current window. The number given is 0 for vertical, 1 for horizontal, and 3 for full width.
move window x y
Moves the window to an absolute position. The frame is not accounted for.
open window
Unhides and raises the window.
options options
Sets the options given.
raise window
Raises the window.
raiseorlower window
Toggles the topness of the window.
rmove window x y
Move a window relative to its current position.
unhide window
Unides the window.
unmaximize window number
Unmaximizes the current window. The number given is 0 for vertical, 1 for horizontal, and 3 for full width.
warp window
Jump to the window.
wchannel window number
Warps a window to the given channel.

Note that the first parameter given when a script executed via keybinds is the current window ID.
